# appTest
Prueba Ubiquo

La app es un proyecto para listar usuarios y crear nuevos.

Se utilizó Ionic como framework para desarrollo de la aplicación, este va muy bien con cordova, es uno de los mejores para desarrollos de app híbridas y su arquitectura es muy buena cuando se piensa en modularizar un proyecto para que sea fácilmente escalable y se pueda reutilizar gran parte de su código. 

Para almacenar la información se utilizó Firebase, es una buena herramienta para almacenar información de forma ágil, esta funciona como una consulta a un webservices y retorna un json.

En cuanto a la arquitectura se crearon templetes para separar el el html, se creo una controlador para manejar los datos consultados a la base de datos y para almacenar los nuevos usuarios. Se tiene un archivo app.js que es el encargado de manejar las dependencias de la app. 

Aun estoy em proceso de aprendizaje con Angular, pero me ha gustado mucho, tiene una comunidad muy grande y fuerte detrás, y las distintas librerías que le integran hacen el trabajo del desarrollar mas ágil, mas limpio y de calidad.

Enlace a la base de datos: https://angelatest.firebaseio.com/
